SOBRE MÍ

En su tiempo me dediqué a la sociología pero con el tiempo descubrí que mi pasión estaba en la informatica . Ahora me dedico a ser estudiante de desarrollo de aplicaciones. Creando a través del código.

PORTFOLIO

Proyecto 1: Sitio Web Personal

Creé un sitio web personal utilizando HTML, CSS y JavaScript para mostrar mi portfolio y habilidades. El sitio incluye una sección de proyectos, un blog y una página de contacto.

Proyecto 2: Aplicación de Tareas

Desarrollé una aplicación de tareas utilizando React para ayudar a los usuarios a organizar sus actividades diarias. La aplicación permite crear, editar y eliminar tareas, así como marcar tareas como completadas.

Proyecto 3: Juego de Memoria

Programé un juego de memoria utilizando JavaScript puro. El juego consiste en encontrar pares de cartas iguales en el menor tiempo posible. Incluye diferentes niveles de dificultad y un sistema de puntuación.

Proyecto 4: Blog Personal

Creé un blog personal utilizando WordPress para compartir mis pensamientos y experiencias sobre el desarrollo web. El blog incluye categorías, etiquetas y un sistema de comentarios para fomentar la interacción con los lectores.
